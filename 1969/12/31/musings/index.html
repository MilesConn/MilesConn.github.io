Stay tuned y'all I have some exciting stuff coming up. Recently I've been reading this book Quantum Computing Since Democritus by Scott Aaronson (amazing blog name might I add). If you were interested in the physics or computing hardware necessary for quantum computing this book isn't for you. But if you wanted to learn that NP doesn't stand for not-polynomial then you might enjoy this book.

One takeaway I want to write about when I'm done is about simulation theory. George Hotz is not the only person who's into simulation theory (see that guy named Musk) but I think he's certainly one of the smarter ones in the pack. I'm just wondering if he's considered if simulation theory is even computable. I think if we look at the Bekenstein Bound that might provided some evidence into the possibility of simulation theory.
